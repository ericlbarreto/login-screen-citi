# 💻 Para acessar o projeto siga os seguintes passos:
Clone o repositório
```
git clone https://github.com/ericlbarreto/login-screen-citi.git
```
Abra o arquivo no seu editor de código e vá para a pasta client:
```
cd client
```
Instale as dependências:
```
npm install
```
Rode a aplicação:
```
npm run dev
```
Abra o link:
```
http://localhost:3000/
```
